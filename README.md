## 🚀 Project Overview

The **Sales Forecasting System** predicts future sales using **Machine Learning** and **Time Series Forecasting** techniques.

The project helps businesses:

✅ Analyze historical sales

✅ Identify sales trends

✅ Forecast future demand

✅ Optimize inventory management

✅ Generate business insights

---

## 🎯 Key Features

### 📊 Sales Trend Analysis
Visualizes daily sales performance over time.

### 📈 Moving Average Analysis
Uses a 7-Day Moving Average to identify trends and smooth fluctuations.

### 📅 Monthly Sales Analysis
Compares sales performance across different months.

### 🤖 Machine Learning Forecasting
Linear Regression predicts future sales patterns.

### ⏳ Time Series Forecasting
ARIMA model forecasts future sales values.

### 📦 Inventory Recommendation
Suggests inventory levels based on future demand.

### 📑 Business Insight Report
Provides:

- Best Performing Month
- Highest Monthly Sales
- Average Daily Sales
- Maximum Daily Sales
- Minimum Daily Sales

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/Sales-Forecasting-System.git
```

### Move into Project Folder

```bash
cd Sales-Forecasting-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Project

```bash
python sales_forecasting.py
```
