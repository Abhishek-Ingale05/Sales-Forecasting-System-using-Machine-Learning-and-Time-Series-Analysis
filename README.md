# Sales-Forecasting-System-using-Machine-Learning-and-Time-Series-Analysis

## 📌 Project Overview
The Sales Forecasting System is a Machine Learning and Time Series Analysis project that predicts future sales using historical sales data.

This project uses:
- **ARIMA Model** for time series forecasting
- **Linear Regression** for additional future sales prediction

The system helps businesses analyze sales trends and make better business decisions.

---

## 🚀 Features
- Load and preprocess sales dataset
- Date conversion and sorting
- Daily sales aggregation
- Sales trend visualization
- Time series forecasting using ARIMA
- Future sales prediction using Linear Regression
- Model evaluation using MAE (Mean Absolute Error)

---

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Statsmodels
